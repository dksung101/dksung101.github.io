Interview Prep with AI

Summary

InnerView is a web application that helps users prepare for interviews by providing feedback on their responses using AI. Users can practice answering questions and receive feedback on their responses. InnerView uses the Gemini Pro API to analyze the user's responses and provide suggested responses and personalized feedback. Most importantly, users can also store their responses as stories to review them later before an upcoming interview.

Tech Stack

  • Frontend: React, DaisyUI, Tailwind CSS
  • Backend/Database: Node.js, Express.js, Firebase Firestore, Firebase Auth
  • AI API: Gemini Pro API
  • Hosting: Heroku
